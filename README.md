# logrion

logrion は、公開されている議論を構造化・整理するためのサービスです。  
主張の正誤や勝敗、人物評価を行うことはありません。

## Purpose

- 議論の論点を可視化する  
- 見落とされがちな前提や未回答点を整理する  
- 第三者にも共有可能な形で構造を示す  

## 公開リポジトリの役割

このリポジトリは Logrion の「公開用の記録」を置くためのものです。  
興味を持った第三者が、プロジェクトの目的・背景・方針・経緯を把握できるようにすることを目的としています。

一方で、サービスの本質に関わる以下の内容はここには掲載しません。

- 実装詳細  
- 抽出ルール  
- スコアリング仕様  
- 具体的手順  
- 運用ノウハウ  

これらは営業秘密・知財・法務リスクを考慮し、別リポジトリで管理します。

- 実装・試行錯誤：`logrion-lab`  
- 設計コンテキスト・草案：`logrion-dev-notes`
