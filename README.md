## Purpose (公開用リポジトリ)

このリポジトリは Logrion の「公開用の記録」を置くためのもの。興味を持った第三者が、プロジェクトの目的・背景・方針・経緯を把握できるようにする。一方で、サービスの本質（実装詳細・抽出ルール・スコアリング・具体的手順・運用ノウハウ等）に関わる内容はここに載せない（営業秘密・知財・法務リスク回避のため）。実装や試行錯誤は logrion-lab、設計コンテキストと草案は logrion-dev-notes に置く。

# logrion

logrion は、公開されている議論を構造化・整理するためのサービスです。
主張の正誤や勝敗、人物評価を行うことはありません。

## Purpose
- 議論の論点を可視化する
- 見落とされがちな前提や未回答点を整理する
- 第三者にも共有可能な形で構造を示す

## Status
This repository is used for design documents and project planning.
Implementation is not included at this stage.
