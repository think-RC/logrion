# logrion Charter（憲章）

logrion は、議論の勝敗や正誤を決めるための存在ではない。
本サービスは、公開されている発言をもとに、
議論の構造を整理し、論点や前提、未回答点を可視化することを目的とする。

logrion は、以下の原則に基づいて運用される。

## 1. 裁定しない
logrion は、いかなる議論においても、
主張の正しさ、優劣、勝敗を判断しない。
人物の評価や人格の推定も行わない。

## 2. 整理に徹する
logrion が行うのは、発言内容の再構成と構造化である。
論点の分解、前提の抽出、未回答点の指摘は行うが、
結論を誘導したり、特定の立場を支持したりすることはない。

## 3. 公開性を尊重する
logrion は、原則として公開されている発言のみを対象とする。
非公開のやり取りや、削除された発言を前提とした分析は行わない。

## 4. 当事者性を侵さない
議論の当事者が自ら求めない限り、
その人物に対する診断、評価、性格分析は行わない。
自己診断は、本人が明示的に要求した場合にのみ提供される。

## 5. 誤解を生まない表現を心がける
出力される内容は、あくまで参考情報であり、
完全性や正確性を保証するものではない。
logrion は、誤解や対立を煽る表現を避け、
第三者にも理解可能な形での提示を目指す。

## 6. 道具であり続ける
logrion は、議論に介入する裁定者ではなく、
思考を整理するための道具である。
利用者が自ら考え、判断する余地を残すことを最優先とする。
